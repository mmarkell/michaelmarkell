Malloc.c

Memory Allocation written in C.
Request Source Code.



Malloc is a project that I built in C as part of a computer systems course. The project utilizes pointer arithmetic and system calls to initialize and maintain a heap of memory. Free blocks are allocated, created, and managed using pointer arithmetic in an efficient freelist.

I wrote entirely from scratch, using a Carnegie Mellon textbook for inspiration. My goal with the project was to make it as efficient as possible, which I managed through a coalescing system. If you have any questions. feel free to shoot me an email!
