CASE 1: None of the cards share characteristics. This can be
represented as $\frac{81}{81}$ $\frac{16}{80}$
$\frac{1}{79}$.\newline
For the first card, we can pick any of the 81 cards. For the
next card we can only choose $\frac{2}{3}$ of each
characteristic which is $2^4$. For the last card we can only
pick 1 card. In this case, $p = .00253$ \newline
CASE 2: The cards have exactly 1 shared characteristic.
Using the same logic from case 1, $\frac{81}{81}$
$\frac{8}{80}$ $\frac{1}{79}$, because now we are keeping 1
characteristic the same, which could be true across any of
the four characteristics. 4*2 cards = 8. \newline
Next, multiply by $3 \choose 1$, because the 1
characteristic we keep the same could be any of them. So the
probability is .0038. \newline
CASE 3: The cards have exactly 2 of the same characteristic.
Using the same logic from case 2, we have we have
$\frac{81}{81} \frac{4}{80} \frac{1}{79}{3 \choose 2}$
\newline
$p = .0019$. \newline
CASE 4: The cards have exactly 3 of the same
characteristics. Using the same logic, we find
$\frac{81}{81} \frac{2}{80} \frac{1}{79} {3 \choose 3}$
\newline
p =.00032
\newline Summing these probabilities finds .00143. Plugging
this into our equation and dividing by p(set), which is
$\frac{1}{79}$, we find .11297. which is the probability
that, given a set of the first three elements, no color is
shared.
